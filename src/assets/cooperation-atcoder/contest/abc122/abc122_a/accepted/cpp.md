# AtCoder Beginner Contest 122
## A - Double Helix
```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string b;
    cin >> b;

    if (b == "A") { cout << "T" << endl; }
    if (b == "C") { cout << "G" << endl; }
    if (b == "G") { cout << "C" << endl; }
    if (b == "T") { cout << "A" << endl; }
}
```
