# AtCoder Beginner Contest 115
## A - Christmas Eve Eve Eve
```cpp
#include <iostream>

using namespace std;

int main() {
    int d;
    cin >> d;

    if (d == 22) { cout << "Christmas Eve Eve Eve" << endl; }
    if (d == 23) { cout << "Christmas Eve Eve" << endl; }
    if (d == 24) { cout << "Christmas Eve" << endl; }
    if (d == 25) { cout << "Christmas" << endl; }
}
```
